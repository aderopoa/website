# Personal Website

My personal website built with React, TypeScript, and Tailwind CSS. Visit at [ayokunle.com](https://ayokunle.com).

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- GitHub Actions (CI/CD)

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
